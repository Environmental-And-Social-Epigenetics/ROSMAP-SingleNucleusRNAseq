# Archived Scripts

These scripts are from earlier iterations of the processing pipeline and have
been superseded by the current 3-stage pipeline in `Processing/Tsai/Pipeline/`.

They are kept here for reference only and are **not** part of the active
pipeline. These scripts contain hardcoded paths from the original development
environment and are **not portable** to other clusters.

- `QC/` — Legacy per-sample QC scripts (DoubletFinder, Scrublet, outlier removal)
- `Batch_Correction/` — Legacy batch correction and clustering scripts
