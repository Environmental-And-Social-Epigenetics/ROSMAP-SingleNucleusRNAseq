# Config

Configuration for the standalone CellBender pipeline.

## Files

### `cellbender_config.sh`

Central configuration file sourced by all scripts in this pipeline. It:

1. **Sources `config/paths.sh`** from the repository root to inherit global paths and environment variables.
2. **Defines CellBender-specific parameters** including:
   - Input/output paths (Cell Ranger output directory, CellBender scratch, final permanent storage)
   - References to the Cell Ranger pipeline's tracking files (`batch_assignments.csv`, `cellranger_completed.txt`)
   - CellBender runtime settings: conda environment path, false positive rate, CUDA toggle
   - SLURM resource allocations (time, CPUs, memory, GPU)
   - Helper function `ensure_pipeline_dirs` to create required directories

## Customization

**Do not edit this file for path changes.** Instead, edit:
- `config/paths.sh` (repository root) for shared path definitions
- `config/paths.local.sh` (repository root, git-ignored) for machine-specific overrides

Edit `cellbender_config.sh` only for CellBender-specific parameters such as FPR, CUDA settings, or SLURM resource requests.
