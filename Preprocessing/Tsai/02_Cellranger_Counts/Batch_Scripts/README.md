# Batch Scripts

Auto-generated SLURM scripts for Cell Ranger and CellBender processing.

Generated by `Scripts/generate_batch_scripts.py`. Do not edit manually.

## Structure

Each `batch_N/` directory (1-16) contains:
- `cellranger/` — Per-patient Cell Ranger SLURM scripts (`{projid}_cellranger.sh`)
- `cellbender/` — Per-patient CellBender SLURM scripts (`{projid}_cellbender.sh`)

Each batch contains ~30 patients. See the parent README for batch processing details.
