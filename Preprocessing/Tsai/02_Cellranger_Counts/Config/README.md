# Config

Pipeline-specific configuration for the Cell Ranger + CellBender batch processing pipeline.

## Files

### `cellranger_config.sh`

Central configuration file that is sourced by every script in the pipeline. It performs two things:

1. **Sources `config/paths.sh`** from the repository root (four levels up) to inherit all global paths and environment variables (e.g., `REPO_ROOT`, `TSAI_FASTQS_DIR`, `CELLRANGER_PATH`, `CELLBENDER_ENV`).
2. **Defines pipeline-specific defaults** including:
   - Batch size and number of batches
   - Cell Ranger parameters (`--no-bam`, `--include-introns`, `--nosecondary`, `--r1-length=26`)
   - CellBender parameters (expected cells, total droplets, FPR, epochs, CUDA)
   - SLURM resource allocations for both Cell Ranger and CellBender jobs
   - Input/output directory paths derived from the global configuration
   - Helper functions (`ensure_pipeline_dirs`, `get_batch_number`, `get_batch_patients`)

## Customization

**Do not edit this file directly for path changes.** Instead, edit:
- `config/paths.sh` (repository root) for shared path definitions
- `config/paths.local.sh` (repository root, git-ignored) for machine-specific overrides

Only edit `cellranger_config.sh` if you need to change pipeline-specific parameters such as batch size, Cell Ranger flags, CellBender hyperparameters, or SLURM resource requests.
